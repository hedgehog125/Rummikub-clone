 So far we've covered:
• Importing the library.
• Making the canvas.
• Adding a sprite.
• Seting the costume of a sprite.

But if your going to make games you need to be able to move your character.
There are 7 different movement commands:


// In javascript is a comment and will not run as code.
Remember: In javascript X 0 Y 0 is the top left hand corner. X + = Move right
Y + = Move down
X - = Move left
Y - = Move up
Note: None of these will work like this, they need to be called by newResetScript or newScript.
